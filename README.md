# Snake Game

A ideia do projeto foi implementar um Snake Game, conhecido popularmente como Jogo da Cobrinha. O jogo consiste em controlar uma cobra que cresce conforme o número de pontos coletados, que aparecem aleatoriamente pelo mapa, evitando colidir com as paredes. O objetivo simular o jogo através de uma matriz de Leds, controlando a cobra com o Joystick.
